Bagaimana cara bergabung dengan server?

Untuk bergabung, Kamu perlu mengisi formulir pendaftaran terlebih dahulu. Setelah disetujui, IP server akan dikirimkan via DM Discord atau Whatsapp atau media sosial lain nya.

Server mendukung versi Minecraft apa saja?

Kami mendukung baik Minecraft Java Edition maupun Bedrock Edition dari 1.8x sampai 1.21x

Apakah server gratis?

Ya, server kami sepenuhnya gratis untuk dimainkan. Donasi bersifat sukarela untuk membantu biaya operasional.

Bagaimana cara mendapatkan bantuan dalam game?

Gunakan command /help dalam game atau hubungi saya (@dipta14) melalui Discord.

Bantuan Teknis

Masalah Koneksi

  • Pastikan koneksi internet stabil
  • Restart game Minecraft
  • Pastikan menggunakan IP yang benar

Masalah Lag

  • Kurangi render distance
  • Matikan shaders dan resource pack berat
  • Pastikan komputer atau handphone memenuhi spesifikasi minimum
  • Tutup aplikasi lain yang berjalan

Error Client

  • Update Minecraft ke versi terbaru
  • Install ulang Minecraft
  • Pastikan Java terinstall dengan benar (Java Edition)
  • Cek firewall dan antivirus
