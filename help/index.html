Frequently Asked Questions

How to join the server?

All new players must complete a Trial Phase before receiving full access. The Trial requires logging into the server for 7 consecutive days (7-day streak).

Which Minecraft versions are supported?

We support both Minecraft Java Edition and Bedrock Edition from version 1.8x to 1.21x

When the server world will be reseted?⭐

We only reset the world when there is an update that added new structure to the game

Is the server free?

Yes, our server is completely free to play. Donations are voluntary to help with operational costs.

How to Register and Login⭐

To register just type this command /register (your password) (confirm your password)

To login simply use command /login (your password that you created with /register)

How to get help in-game?⭐

Use the /help command in-game or contact me (@dipta14) via Discord.

Server Commands⭐

  • /clan Create or manage your clan
  • /nv Get night vision effect
  • /skin Change your skin (use custom skin on https://warlordrealm.ct.ws/skin)
  • /skins Open skin GUI
  • /spawn Teleport to world spawn point
  • /changepass (old password) (new password) Change your account password on the server
  • /pwc Change player locator dot color
  • /playtime View your total playtime
  • /playtime top View top 10 most active players
  • /expbottle Convert your experience into bottles
  • /imageframe Create images using maps
  • /trash Open disposal gui
  • /ping Check your network latency
  • /daily Claim Daily,Weekly,Monthly Rewards
  • /msg (player) (message) Send a private message
  • /r Reply to the last private message
  • /leaderboard Displays the top players based on server statistics.
  • /sb Toggle scoreboard
  • /link Link your Minecraft account to WarGuard Discord bot
  • /daycount View how long the server's world has been running
  • /discord View the Discord server invite link
  • /mobhealt Toggle mob health hologram display
  • /email Send in-game mail
  • /interactivechat list View available chat placeholders
  • /emoji View all emojis available on the server
  • /kill Kill yourself with a custom death message
  • /suicide Same as /kill
  • /ivtoggle Toggle Interaction Visualizer display
  • /scary Get a cursed scary pumpkin head
  • /mention Toggle chat mentions on or off

List of placeholders

  • [item] Show the Item you are holding in the chat
  • [inv] Show your Inventory in the chat
  • [ender] Show your Ender Chest in the chat
  • [gametime] Shows the current time of the world you are in
  • [pos] Shows your current position
  • [ping] Shows your current ping

Terms of Service⭐

Mandatory Trial Phase (7-Day Login Streak)

All new players must log in for 7 consecutive days. Missing even one day will automatically fail the Trial.

If the 7-day streak is broken, the Trial Phase will be closed immediately and marked as rejected.

Trial Re-Request Policy

Players who fail the Trial may request the Admin to reopen and repeat the Trial Phase. Approval to restart the Trial is not guaranteed and remains fully at staff discretion.

Authority & Enforcement

By playing on Warlord Realm, you agree to follow all server rules. Staff reserve the full right to take action at any time, including:

  • Kick
  • Temporary or Permanent Ban
  • Player Data Reset
  • Rank Removal

All staff decisions are final and non-negotiable.
