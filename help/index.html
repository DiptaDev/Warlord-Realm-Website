Frequently Asked Questions

How to join the server?

To join, you need to fill out a registration form first. Once approved, the server IP will be sent via DM on Discord, WhatsApp, or other social media.

Which Minecraft versions are supported?

We support both Minecraft Java Edition and Bedrock Edition from version 1.8x to 1.21x

When the server world will be reseted?⭐

We only reset the world when there is an update that added new structure to the game

Is the server free?

Yes, our server is completely free to play. Donations are voluntary to help with operational costs.

How to Register and Login⭐

To register just type this command /register (your password) (confirm your password)

To login simply use command /login (your password that you created with /register)

How to get help in-game?⭐

Use the /help command in-game or contact me (@dipta14) via Discord.

Server Commands⭐

  • /clan Create or manage your clan
  • /nv Get night vision effect
  • /skin Change your skin (use custom skin on https://warlordrealm.ct.ws/skin)
  • /changepass (old password) (new password) Change your account password on the server
  • /pwc Change player locator dot color
  • /playtime View your total playtime
  • /playtime top View top 10 most active players
  • /expbottle Convert your experience into bottles
  • /imageframe Create images using maps
  • /ping Check your network latency
  • /msg (player) (message) Send a private message
  • /r Reply to the last private message
  • /link Link your Minecraft account to WarGuard Discord bot
  • /discord View the Discord server invite link
  • /mobhealt Toggle mob health hologram display
  • /email Send in-game mail
  • /interactivechat list View available chat placeholders
  • /emoji View all emojis available on the server

Technical Support

Connection Issues

  • Make sure your internet connection is stable
  • Restart Minecraft
  • Make sure you're using the correct IP

Lag Issues

  • Reduce render distance
  • Disable shaders and heavy resource packs
  • Make sure your computer or phone meets minimum specifications
  • Close other running applications

Client Errors

  • Update Minecraft to the latest version
  • Reinstall Minecraft
  • Make sure Java is installed correctly (Java Edition)
  • Check firewall and antivirus settings
