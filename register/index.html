Welcome to the Warlord Realm Vanilla Survival Multiplayer server registration form!

We have reopened our vanilla survival server for minecraft players who want to chill, virtually touch grass, and play casually! (*or for those who want to explore together and also CREATE WAR).

You can play survival quite comfortably here with other members from our community (100% real, no fake). This server doesn't use mods that significantly affect vanilla gameplay, which means the playing experience is pretty much similar to regular survival.

Registration is open for Minecraft Java or Bedrock players who have premium or cracked accounts. The server uses the latest game version, which is 1.21.x (the server can be joined by versions 1.8x to 1.21.x)

Make sure you have a Discord account and have joined the Warlord Network Discord server before registering! Click to join the Warlord Network Discord server
