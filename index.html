Warlord Realm

Minecraft Server - Java & Bedrock

Tentang Warlord Realm

Warlord Realm adalah server chill yang lumayan santai, cocok untuk player yang bermain santai dan terkadang akan ada konflik antar clan akibat kebodohan rafif. sebelum nya server ini hanya di peruntukan untuk teman satu sircle tapi karena teman teman saya membagikan ip ke orang orang sekarang server ini jadi banyak pemain

Multiplatform

Dukung Minecraft Java dan Bedrock Edition. Bermain dengan teman Anda terlepas dari platform yang digunakan.

Komunitas Santai

Komunitas yang agak friendly dan santai, cocok untuk bermain setelah lelah beraktivitas seharian.

Discord Community

Bergabung dengan Discord kami untuk update terbaru, berita server, dan berinteraksi dengan komunitas.

Cara Bergabung

Proses Bergabung

Untuk bergabung dengan server kami, Kamu perlu mengisi formulir pendaftaran terlebih dahulu. Setelah formulir disetujui, IP server akan dikirimkan kepada Kamu melalui DM Discord,Whatsapp atau media sosial lainnya dan akan di berikan role khusus di discord server agar bisa mengakses channel berita server dan update.

Isi Formulir Pendaftaran

Pastikan Kamu telah bergabung dengan Discord server kami untuk memudahkan komunikasi.
