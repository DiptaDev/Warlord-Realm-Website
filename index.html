About Warlord Realm

Warlord Realm is a relatively relaxed and calm server, suitable for casual players. Sometimes there is a conflicts between clans and resulting a war to happend. Originally this server was intended for a single group of friends ship, but some friends shared the server ip address with others popple, resulting a lot of new members
